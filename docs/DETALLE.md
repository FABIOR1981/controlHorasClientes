# Documentación detallada de controlHorasClientes

## Descripción general
controlHorasClientes es una aplicación web diseñada para gestionar y registrar las horas trabajadas por clientes, permitiendo un control eficiente y transparente de la información laboral. El sistema facilita la administración de usuarios, clientes y reportes de horas, integrando funcionalidades de autenticación y visualización de datos.

## Características principales
- Registro y edición de horas trabajadas por cliente.
- Gestión de usuarios y clientes.
- Visualización de informes y reportes.
- Interfaz intuitiva y adaptable.
- Seguridad mediante autenticación.

## Estructura del proyecto
- **index.html**: Página principal de la aplicación.
- **css/**: Estilos visuales para la interfaz.
- **js/**: Lógica y funcionalidades del sistema.
- **data/**: Archivos de datos (JSON) para usuarios, clientes y horas.
- **html/**: Vistas secundarias y módulos.
- **docs/**: Documentación técnica y de usuario.

## Instalación y uso
1. Clonar el repositorio.
2. Abrir `index.html` en un navegador web.
3. Configurar los archivos de datos según las necesidades.
